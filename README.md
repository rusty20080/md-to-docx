# Markdown to Word Converter

A simple web application to convert Markdown files (.md) to Microsoft Word documents (.docx) using Pandoc. Perfect for teams that need to share markdown content in Word format.

## Features
- Drag-and-drop or file-select interface
- Real-time conversion status updates
- Error handling for invalid files/conversions
- Cross-platform (Windows/macOS/Linux) via web browser
- Uses your existing Pandoc installation

## Prerequisites
1. **Pandoc** must be installed: [Download Pandoc](https://pandoc.org/installing.html)
2. Python 3.8+ (with pip)
